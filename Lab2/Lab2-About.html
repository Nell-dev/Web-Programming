About Lab 2

In Lab 2, we learned how to create both absolute and relative links, structure HTML tables with captions, and use basic native form controls such as text inputs, checkboxes, and radio buttons. We also explored how to organize and build complete web forms using HTML5 input types. Finally, we practiced debugging HTML to ensure clean, well-structured code.
